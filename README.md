# Booking
图书馆预约
下载到本地把CodeDetect放在Booking下,运行Runs.py就可以预约了
如果不想自己改代码的话那建议先手动预约一次,再取消.想要预约的座位就到了常用座位里
然后可以获取一下目标座位的Xpath路径.以字符串的格式填入参数就可以顺利运行了,需要修改的参数都在Runs.py中
